# java-cli-bloop-mill-hsqldb-simple

## Description
A java millbuild build, that connects to hsqldb
database.

## Tech stack
- java
- bloop
- millbuild
- hsqldb driver

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
